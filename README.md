# Xeyronox Link Bot

A simple Telegram bot that shares Xeyronox's social profiles, tool shop, and portfolio links.

---

## Bot Info

- **Bot Name:** Xeyronox Link Bot  
- **Bot Username:** @XeyronoxLinkBot  *(replace with your actual bot username)*

---

## Features

- Commands:
  - `/start` — Welcome message  
  - `/help` — Help instructions  
  - `/links` — Official social and tool shop links  
  - `/shop` — Link to hacking tool shop  
  - `/portfolio` — Portfolio section (coming soon)  
  - `/language` — Language selection (English only currently)  

- Inline buttons for links  
- User-friendly alerts for upcoming features  
- Handles unknown commands gracefully  

---

## Requirements

- Python 3.10 or higher  
- `python-telegram-bot` v20+  
- `python-dotenv` for environment variables
